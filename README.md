# NIEM Forensics
The [NIEMOpen Forensics Subcommittee](https://niemopen.org/community/forensics/) was established in 2025 after identifying the need for a unified forensic data framework. The Forensics Subcommittee is responsible for updating and maintaining the forensics domain elements found in the [NIEMOpen](https://niemopen.org/) data model, leading to a common vocabulary for exchanging forensics information and increased interoperability across organizations.

## Tiger Teams
The Forensics Subcommittee features 6 expert groups, or “Tiger Teams”, that are responsible for refining and adding data elements and definitions for the following disciplines:
*	Chemistry
*	Document and Media Exploitation
*	Electronics Exploitation
*	Firearms and Toolmarks
*	Uncrewed Systems
*	Weapons Technical Intelligence/Weapons Exploitation
